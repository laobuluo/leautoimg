# LeAutoImg WordPress 插件

## 插件简介
LeAutoImg 是一个简单而实用的 WordPress 插件，用于自动为没有图片的文章添加随机预设图片。这个插件可以帮助您保持博客文章的视觉吸引力，确保每篇文章都有配图。

## 主要功能
- 自动检测文章是否包含图片
- 为无图文章自动插入随机预设图片
- 支持自定义图片插入位置（第1-5段后）
- 支持图片对齐方式设置（左对齐/居中）
- 支持自定义 CSS 样式
- 支持多张预设图片随机选择
- 集成 WordPress 媒体库，方便上传图片

## 安装方法
1. 下载插件压缩包
2. 在 WordPress 后台进入"插件 > 安装插件"
3. 点击"上传插件"按钮
4. 选择下载的压缩包文件
5. 点击"现在安装"
6. 安装完成后点击"启用插件"

## 使用方法
1. 在 WordPress 后台进入"设置 > LeAutoImg"
2. 勾选"启用插件"选项
3. 添加预设图片（支持多张）
4. 设置图片插入位置
5. 选择图片对齐方式
6. 可选：添加自定义 CSS 样式
7. 点击"保存更改"

## 设置说明
### 启用插件
- 勾选此选项以启用自动图片插入功能

### 预设图片
- 可以添加多张图片作为预设
- 支持通过媒体库上传图片
- 支持手动输入图片 URL

### 插入段落
- 可选择在第1-5段后插入图片
- 如果文章段落数少于选择的位置，图片将插入在文章末尾

### 图片对齐
- 左对齐：图片靠左显示
- 居中：图片居中显示

### 自定义 CSS
- 可以添加自定义 CSS 样式来调整图片外观
- 例如：`max-width: 100%; margin: 20px 0;`

## 注意事项
- 插件只会为没有图片的文章添加预设图片
- 如果文章已经包含图片，插件不会进行任何修改
- 建议使用适当大小的图片以确保良好的加载性能
